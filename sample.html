How to Write a Markdown Post

2025-07-15 튜토리얼

여기에 본문 내용이 들어갑니다. 이 주석 아래에 마크다운 형식으로 글을 작성하시면 됩니다. 본문 내용은 자유롭게 마크다운 문법을 사용하여 작성할 수 있습니다.

마크다운 작성 시 주의사항

  • 제목 태그는 ## 부터 시작하세요 (# 대신 메타데이터를 사용)
  • 코드 블록은 ```언어명 으로 시작하고 ``` 으로 끝내세요
  • 이미지는 상대 경로로 링크하세요 (예: )

참고: 이제 메타데이터 주석에 title을 지정하면 포스트 헤더에 제목으로 표시됩니다.
