가이드

다양한 옵션으로 안전한 비밀번호를 생성하세요. 길이, 특수문자, 숫자, 대문자 설정을 통해 원하는 조건의 비밀번호를 만들 수 있습니다.

여기에 생성된 비밀번호가 표시됩니다

비밀번호 강도: 아직 생성되지 않음

안전한 비밀번호 사용 팁

  • 12자 이상의 비밀번호를 사용하세요.
  • 다양한 종류의 문자(대문자, 소문자, 숫자, 특수문자)를 조합하세요.
  • 여러 서비스에 동일한 비밀번호를 사용하지 마세요.
  • 비밀번호 관리 도구를 사용하여 복잡한 비밀번호를 안전하게 보관하세요.
  • 정기적으로 비밀번호를 변경하는 습관을 가지세요.

현재 사용 중인 비밀번호의 강도를 확인하세요. 비밀번호를 입력하면 보안 점수와 안전성 기준 충족 여부를 확인할 수 있습니다.

비밀번호 강도: 아직 입력되지 않음

보안 기준 충족 여부

  • ❌ 최소 12자 이상 (길이가 길수록 더 안전)
  • ❌ 대문자 포함 (A-Z)
  • ❌ 소문자 포함 (a-z)
  • ❌ 숫자 포함 (0-9)
  • ❌ 특수문자 포함 (!@#$%^&* 등)
  • ❌ 연속된 문자 3개 이상 없음

안전한 비밀번호로 변경하기

검증 결과가 만족스럽지 않다면, '생성기' 탭으로 이동하여 더 강력한 비밀번호를 만들어보세요.

비밀번호 생성기 사용법

  1. 비밀번호 길이 슬라이더를 사용하여 원하는 길이로 설정합니다 (6-32자).
  2. 비밀번호 옵션을 선택합니다:
    • 대문자 포함: 'A-Z' 범위의 영문 대문자를 포함합니다.
    • 소문자 포함: 'a-z' 범위의 영문 소문자를 포함합니다.
    • 숫자 포함: '0-9' 범위의 숫자를 포함합니다.
    • 특수 문자 포함: '!@#$%^&*()' 등의 특수 문자를 포함합니다.
    • 비슷한 문자 제외: '1', 'l', 'I', '0', 'O' 등 혼동되기 쉬운 문자를 제외합니다.
  3. "비밀번호 생성하기" 버튼을 클릭하여 결과를 생성합니다.

비밀번호 결과 해석

  • 비밀번호 강도: 생성된 비밀번호의 보안 수준을 나타냅니다.
  • 약함 (빨간색): 길이가 짧거나 문자 종류가 단일 유형인 비밀번호입니다.
  • 중간 (주황색): 최소 8자 이상이며 2가지 이상의 문자 유형을 포함합니다.
  • 강함 (녹색): 최소 10자 이상이며 3가지 이상의 문자 유형을 포함합니다.
  • 매우 강함 (진한 녹색): 최소 12자 이상이며 모든 문자 유형을 포함합니다.

보안 비밀번호 팁

  • 다양성: 대문자, 소문자, 숫자, 특수문자를 모두 포함하세요.
  • 길이: 최소 12자 이상의 비밀번호를 사용하세요.
  • 고유성: 각 계정마다 서로 다른 비밀번호를 사용하세요.
  • 정기 변경: 비밀번호를 3-6개월마다 변경하세요.
  • 관리 도구: 비밀번호 관리자를 사용하여 복잡한 비밀번호를 안전하게 보관하세요.
