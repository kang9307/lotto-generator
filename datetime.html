시간/날짜 계산기 사용법

날짜 차이 계산

두 날짜 사이의 기간을 계산하는 기능입니다.

  1. 시작 날짜와 종료 날짜를 선택합니다.
  2. '계산하기' 버튼을 클릭합니다.
  3. 두 날짜 사이의 일수, 주수, 개월수, 년수가 표시됩니다.

날짜 더하기/빼기

특정 날짜에 일정 기간을 더하거나 뺄 수 있는 기능입니다.

  1. 기준 날짜를 선택합니다.
  2. 더하기 또는 빼기 연산을 선택합니다.
  3. 더하거나 뺄 기간을 숫자와 단위(일, 주, 개월, 년)로 지정합니다.
  4. '계산하기' 버튼을 클릭하면 계산된 새 날짜가 표시됩니다.

업무일 계산

주말을 제외한 업무일 수를 계산하는 기능입니다.

  1. 시작 날짜와 종료 날짜를 선택합니다.
  2. 주말 제외 여부를 선택합니다(기본값: 예).
  3. '계산하기' 버튼을 클릭하면 업무일 수와 주말 수가 표시됩니다.

시간 변환

초, 분, 시간, 일 단위 사이의 변환 기능입니다.

  1. 변환할 시간 값을 입력합니다.
  2. 변환 전 단위와 변환 후 단위를 선택합니다.
  3. '변환하기' 버튼을 클릭하면 변환된 결과가 표시됩니다.

자주 묻는 질문 (FAQ)

Q: 날짜 차이 계산시 시작일과 종료일을 포함하나요?

A: 예, 시작일과 종료일을 모두 포함합니다. 예를 들어, 1월 1일부터 1월 3일까지는 3일로 계산됩니다.

Q: 업무일 계산에서 공휴일도 제외할 수 있나요?

A: 현재 버전에서는 토요일과 일요일만 제외할 수 있습니다. 공휴일 제외 기능은 향후 업데이트에서 추가될 예정입니다.

Q: 월 단위 계산은 어떻게 이루어지나요?

A: 월 단위 계산은 날짜의 실제 월 차이를 기준으로 합니다. 예를 들어, 1월 15일에서 2월 15일까지는 정확히 1개월로 계산됩니다. 단, 일 수가 다를 경우(예: 1월 31일에서 2월 28일) 적절히 조정됩니다.

Q: 윤년(leap year)은 어떻게 처리되나요?

A: 모든 계산에서 윤년은 자동으로 고려됩니다. 2월 29일이 존재하는 윤년에는 해당 날짜도 정확히 계산에 포함됩니다.
